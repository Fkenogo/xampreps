## Image Extraction Notes

1. **Source:** Extract diagrams from `PLE MATHS 2015-CLEAN2.pdf`
2. **Format:** PNG with transparent background preferred
3. **Resolution:** Minimum 300 DPI for clarity
4. **Dimensions:** Approximately 400-600px wide for web display
5. **File naming:** Use lowercase with hyphens for URL compatibility

## Upload Instructions

1. Upload each image to Firebase Storage under the path shown
2. Verify the download URL is accessible
3. Update the JSON import file if URLs differ from the placeholder paths
4. Test each image displays correctly in the exam preview
